splunk documentation search

Working on a project that relies heavily on Splunk (that is, Splunk itself) requires frequent trips to the Splunk Documentation.

I'm a LaunchBar user (think Quicksilver), here's the steps I've used to create a Custom Search Template for Splunk Documentation:

  1. Open the LaunchBar Index (Option+Command+I).
  2. Select the Custom Search Template folder.
  3. On the bottom of the LaunchBar Index window click Add....
  4. For Name, enter a string you'll use to shortcut to this search, for Splunk Documentation I used docs.
  5. For Details, enter: http://docs.splunk.com/Special:SplunkSearch/docs?q=*
  6. Save (Command+S).

To search:

  1. Hit your LaunchBar shortcut key sequence (for me it's Command+Space).
  2. Type 'docs' + Space.
  3. Enter your search terms.
  4. Press [return].

Enjoy!
