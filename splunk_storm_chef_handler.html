splunk storm chef handler

Using my Chef Exception & Reporting Handler for Splunk Storm you can easily search for and report on your chef-client runs.

Steps

  1. Create a Splunk Storm account.
  2. Retrieve your Splunk Storm REST API Credentials.
  3. Download the chef_handler Cookbook.
  4. Add the chef_handler Cookbook to your Run List.

  5. Given you've retrieved your Access Token as ACCESS_TOKEN and Project ID as PROJECT_ID, add a chef_handler Resource to one of your Recipes:

    chef_handler 'SplunkStorm::SplunkStormHandler' do
  action :enable
  source File.join(node['chef_handler']['handler_path'], 'splunkstorm.rb')
  arguments ['ACCESS_TOKEN', 'PROJECT_ID']
end
